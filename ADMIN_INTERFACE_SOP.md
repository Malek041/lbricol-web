# SOP: Accessing and Using the Admin Interface

This Standard Operating Procedure (SOP) outlines the steps to access the Admin interface and the features available within it.

## 1. How to Access the Admin Portal

The Admin interface is protected by a secret code. Follow these steps to log in:

1.  **Open Profile**: On the mobile app, tap on the **Profile** tab in the bottom navigation.
2.  **Locate 'Admins'**: Scroll down the menu items to find the **"Administration"** (fr: "Administration") option with the shield icon.
3.  **Enter Secret Code**:
    -   A numeric keypad/modal will appear requesting a secret code.
    -   Enter the master code: `2026LB`.
    -   Tap **"Access"**.
4.  **Admin Mode**: Once verified, you will be switched to **Admin Mode**. The bottom navigation will change to show Admin-specific tabs (Performance, Orders, etc.).

## 2. Key Features

### A. Admin Dashboard (Performance)
-   **Aggregated Metrics**: View total jobs, revenue, and growth metrics.
-   **City Filtering**: Filter the performance data by specific cities (e.g., Casablanca, Marrakech).
-   **Trends**: Visual representation of order volume and provider activity.

### B. Admin Orders View
-   **Global Order Tracking**: See all orders submitted across the platform.
-   **Status Management**: Manually update order statuses if necessary.
-   **City Filters**: Easily switch between cities to manage regional operations.

### C. Create Bricoler Profile
-   **Manual Onboarding**: Bypass the standard onboarding for specific premium providers.
-   **Meta ID & Custom Data**: Add custom Meta IDs, CIN, and specific bio/pitch directly into the system.
-   **Location Assignment**: Assign providers to specific cities and work areas.

## 3. How to Exit Admin Mode

1.  Go to the **Profile** tab.
2.  Tap on **"Exit Admin Mode"** (fr: "Quitter le mode Admin").
3.  The app will revert to the standard Client/Provider interface.

---
**Security Note:** Do not share the master admin code `2026LB` with anyone outside of the core management team.
